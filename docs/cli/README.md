# CLI

Instalar [Node.js](https://nodejs.org/es/download/)

    $ node -v
    $ npm -v # yarn

Instalar vue-cli

    $ npm i -g @vue/cli
    $ vue -V

## Crear un proyecto

    $ vue create my-app
    $ npm run serve
    $ npm run build
